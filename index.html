Dairy TMR Calculator

How to Install (iPhone & Android)

iPhone: Tap the green button → Share icon → "Add to Home Screen"

Android: Tap the green button → "Install app"

After installing, open from your home screen icon for the best experience.

How to Use the TMR Calculator

1. Set up Groups & Rations (one-time)
Tap Groups & Rations tab → + ADD GROUP → name it (e.g., High Group)
Then + Add Ration under the group (e.g., Standard)

2. Add ingredients
Go to Calculator tab → select Group & Ration → tap + Add Ingredient → type feed name

3. Make a daily batch
Select Group → Ration → enter number of cows → fill lbs per cow → GENERATE BATCH SHEET

4. Print
Tap PRINT SHEET for a clean one-page loading order (bold text, cumulatives, total)

Backup
At bottom → "Backup My Data" (download file) or it's auto-saved to cloud when logged in

That's it — daily batches in under a minute!
