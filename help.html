How to Use the TMR Calculator

1. Set Up Groups & Rations (one-time)

Tap the Groups & Rations tab

• Tap + ADD GROUP → type name (e.g., High Group)

• Under the group, tap + Add Ration → type name (e.g., Standard)

• Go back to Calculator tab → select Group & Ration → tap + Add Ingredient to add feeds

2. Make a Daily Batch

Tap the Calculator tab

• Choose Group

• Choose Ration

• Enter number of cows

• Fill in lbs per cow for each ingredient

• Tap GENERATE BATCH SHEET

3. Print the Loading Order

Tap PRINT SHEET

It prints a clean one-page sheet with bold ingredients, cumulatives, and total weight — perfect for the mixer.

4. Backup

Your data saves to the cloud automatically when logged in.

For extra safety: Tap Tools & Help → Backup My Data → save the file.
